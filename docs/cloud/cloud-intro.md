# ☁️ Cloud Computing

## O que é Cloud Computing?

Cloud Computing (Computação em Nuvem) é a entrega de recursos de computação pela internet sob demanda. Em vez de comprar e manter servidores físicos, você aluga recursos conforme necessário.

**Analogia**: É como usar Netflix em vez de comprar DVDs. Você paga pelo que usa, quando usa, sem se preocupar com a infraestrutura física.

## Principais Modelos de Serviço

### IaaS (Infrastructure as a Service)

Infraestrutura virtualizada sob demanda.

- **Exemplo**: EC2 (AWS), Virtual Machines (Azure)
- **Você gerencia**: SO, runtime, aplicações
- **Provedor gerencia**: Hardware, rede, virtualização

### PaaS (Platform as a Service)

Plataforma completa para desenvolvimento.

- **Exemplo**: App Service (Azure), Elastic Beanstalk (AWS)
- **Você gerencia**: Aplicações e dados
- **Provedor gerencia**: SO, runtime, infraestrutura

### SaaS (Software as a Service)

Software pronto para uso.

- **Exemplo**: Gmail, Office 365, Salesforce
- **Você gerencia**: Seus dados
- **Provedor gerencia**: Tudo mais

## Principais Provedores

- **[AWS](aws/aws-intro.md)**: Amazon Web Services
- **[Azure](azure/azure-intro.md)**: Microsoft Azure

## Quando usar Cloud?

✅ **Use quando**:

- Precisa escalar rapidamente
- Quer reduzir custos iniciais (CapEx)
- Necessita alta disponibilidade
- Foco no produto, não na infraestrutura

❌ **Evite quando**:

- Requisitos de compliance impedem uso de nuvem
- Aplicações legadas muito acopladas a hardware específico
- Custos previsíveis de longo prazo favorecem on-premises

## Conceitos-chave

- **Elasticidade**: Escalar recursos automaticamente
- **Pay-as-you-go**: Pague apenas pelo que usar
- **Alta disponibilidade**: Múltiplas zonas/regiões
- **Disaster Recovery**: Backup e recuperação facilitados
